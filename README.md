It focuses on the JavaScript programming language, syntax, coding standards, documentation, and how to debug programs.  Topics include Functions, Data Types, Operators, Arrays, Controlling Flow, JavaScript events, and Regular Expression.  Projects will use HTML, CSS, and JavaScript to create dynamic web pages.  GitHub is used to host coding projects.

Topics include marketing techniques, business/client needs, business skills, business communication, organization and team development, and leveraging communities of interest.
            </p>

    </tr>
